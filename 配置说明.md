## ⚙️ 工作原理

```
每天 16:00 (cron 触发)
    ↓
send_feishu_auto.py 运行
    ↓
1. 执行 stock_monitor.py 获取股票数据
2. 从 openclaw.json 获取飞书凭证
3. 调用飞书 API 获取 access_token
4. 发送消息到用户飞书
    ↓
完成！
```

## 📞 问题排查

### 没有收到推送

1. **检查日志**:
   ```bash
   cat /tmp/stock_monitor.log
   ```

2. **检查定时任务**:
   ```bash
   crontab -l | grep stock
   ```

3. **手动测试**:
   ```bash
   python3 send_feishu_auto.py
   ```

### 发送失败

查看日志中的错误信息，常见原因：
- 飞书凭证过期 → 重新授权
- 用户 ID 错误 → 检查 config.json
- 网络问题 → 检查网络连接
