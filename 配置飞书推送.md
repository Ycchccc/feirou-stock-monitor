# 配置飞书推送

股票监控支持三种推送方式，选择一种即可：

## 方式一：飞书机器人 Webhook（推荐）

### 步骤

1. **创建飞书机器人**
   - 在飞书中创建一个群（可以只拉自己）
   - 点击右上角「...」→「添加机器人」
   - 选择「自定义机器人」
   - 设置机器人名称（如：股票助手）
   - 复制 webhook URL

2. **保存 webhook URL**
   ```bash
   # 将 URL 保存到配置文件
   echo "https://open.feishu.cn/open-apis/bot/v2/hook/xxx" > ~/.stock_monitor/webhook.txt
   ```

3. **测试**
   ```bash
   python3 daily_with_notify.py
   ```

### 优点
- 简单可靠
- 无需额外配置
- 支持 Markdown 格式

---

## 方式二：OpenClaw 消息工具

在飞书私信中直接运行：

```bash
python3 stock_monitor.py
```

报告会自动发送到当前会话。

### 优点
- 无需配置
- 直接发送到私信

---

## 方式三：查看日志

定时任务会将报告保存到日志文件：

```bash
# 查看今日报告
cat /tmp/stock_monitor.log

# 或查看保存的报告
cat /tmp/stock_report_today.txt
```

---

## 定时任务说明

安装定时任务后，每个工作日 16:00 自动运行：

```bash
# 查看定时任务
crontab -l | grep stock

# 查看定时任务日志
cat /tmp/stock_monitor.log
```

---

## 常见问题

**Q: webhook 发送失败？**
- 检查 webhook URL 是否正确
- 确保机器人已启用
- 测试：`curl -X POST -H "Content-Type: application/json" -d '{"msg_type":"text","content":{"text":"test"}}' https://open.feishu.cn/open-apis/bot/v2/hook/xxx`

**Q: 收不到消息？**
- 检查定时任务是否运行：`cat /tmp/stock_monitor.log`
- 手动运行测试：`python3 daily_with_notify.py`

**Q: 如何修改推送时间？**
- 编辑定时任务：`crontab -e`
- 修改时间（格式：分 时 * * *）
- 例如：`0 17 * * 1-5` 表示工作日 17:00
