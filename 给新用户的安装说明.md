# 📈 飞书股票监控助手 - 安装指南

欢迎使用 **feirou-stock-monitor**！

这是一个自动监控股票的 Skill，每天收盘后自动推送持仓日报到你的飞书。

---

## 🚀 快速安装（3 分钟搞定）

### 方式 A：一键安装脚本（推荐）

```bash
# 1. 运行安装脚本
cd ~/.openclaw/workspace/skills/feirou-stock-monitor
./install.sh

# 2. 修改飞书用户 ID
code ~/.stock_monitor/config.json
# 修改 "user_id": "ou_xxx" 为你的飞书 ID

# 3. 测试
python3 daily_with_notify.py
```

---

### 方式 B：手动安装

```bash
# 1. 安装依赖
cd ~/.openclaw/workspace/skills/feirou-stock-monitor
pip3 install akshare pandas

# 2. 运行配置
python3 quick_setup.py

# 3. 修改飞书用户 ID
code ~/.stock_monitor/config.json
# 修改 "user_id": "ou_xxx" 为你的飞书 ID

# 4. 加载定时任务
launchctl load ~/Library/LaunchAgents/com.stock-monitor.daily.plist

# 5. 测试
python3 daily_with_notify.py
```

## 📋 配置说明

### 修改股票列表

编辑 `~/.stock_monitor/config.json`：

```json
{
  "stocks": [
    {
      "code": "600010",
      "name": "包钢股份",
      "cost": 4.253,
      "shares": 100
    },
    {
      "code": "600519",
      "name": "贵州茅台",
      "cost": 1800.00,
      "shares": 100
    }
  ]
}
```

**字段说明：**
- `code`: 股票代码
- `name`: 股票名称
- `cost`: 你的成本价
- `shares`: 持股数量

---

### 获取飞书用户 ID

**方式 1：问机器人**
在飞书中问机器人："我的用户 ID 是什么？"

**方式 2：查看消息**
从飞书消息中获取（格式：`ou_xxx`）

## ❓ 常见问题

**Q: 收不到日报？**
- 检查定时任务：`launchctl list | grep stock-monitor`
- 查看日志：`cat /tmp/stock_monitor.log`
- 手动运行：`python3 daily_with_notify.py`

**Q: 修改推送时间？**
- 编辑 `~/.stock_monitor/config.json`，修改 `notify_time`
- 重新运行：`python3 quick_setup.py`

**Q: 添加新股？**
- 编辑 `~/.stock_monitor/config.json`，在 `stocks` 数组中添加

**Q: 卸载？**
```bash
launchctl unload ~/Library/LaunchAgents/com.stock-monitor.daily.plist
rm -rf ~/.stock_monitor
```
