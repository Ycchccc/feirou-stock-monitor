# 📦 如何分享 feirou-stock-monitor 给别人

## 快速分享（推荐）

### 1️⃣ 发送文件

将以下文件发送给对方：
```
~/.openclaw/workspace/skills/feirou-stock-monitor-share.tar.gz
```

**文件大小：** 约 16KB

**发送方式：**
- 飞书发送文件
- 微信传输
- 邮件附件

---

### 2️⃣ 发送安装说明

同时发送这段文字：

---

📈 **欢迎使用飞书股票监控助手！**

这是一个自动监控股票的 Skill，每天收盘后自动推送持仓日报到飞书。

**安装步骤（3 分钟搞定）：**

```bash
# 1. 解压
tar -xzf feirou-stock-monitor-share.tar.gz -C ~/.openclaw/workspace/skills/

# 2. 运行一键安装
cd ~/.openclaw/workspace/skills/feirou-stock-monitor
./install.sh

# 3. 修改飞书用户 ID（重要！）
code ~/.stock_monitor/config.json
# 修改 "user_id": "ou_xxx" 为你的飞书 ID
# 如果不知道你的 ID，问机器人帮你查

# 4. 测试
python3 daily_with_notify.py
```

**详细说明请查看：** `给新用户的安装说明.md`

---

### 3️⃣ 对方安装后

告诉对方：
1. 每天 16:00 自动收到股票日报
2. 可以修改 `~/.stock_monitor/config.json` 添加/删除股票
3. 有问题查看 `给新用户的安装说明.md`

## 对方可能需要的帮助

### 获取飞书用户 ID

告诉对方：
- 在飞书中问机器人："我的用户 ID 是什么？"
- 或者查看飞书消息中的 sender_id（格式：ou_xxx）

### 添加股票

编辑 `~/.stock_monitor/config.json`：
```json
{
  "stocks": [
    {"code": "600519", "name": "贵州茅台", "cost": 1800, "shares": 100}
  ]
}
```

### 修改成本价

同样编辑配置文件，修改 `cost` 字段。
